CodeFit_AI.js

Find out if coding fits you — in about five minutes — then learn web development with an AI tutor and a live code sandbox.

Check your fit

How it works

1

Assess

Start with a short fit check — a problem-solving quiz plus a Big Five work-style questionnaire. You get an encouraging read on your strengths and a couple of tech roles that tend to suit you.

2

Learn

The lessons unlock: a beginner curriculum in HTML, CSS, and JavaScript, taught by an AI tutor that explains each topic for your level and sees the code in your sandbox.

3

Track

Topics check off as you complete them, and your progress, saved snippets, and preferences follow you across every device.

What's inside

AI personas, on demand

Switch the AI between a code explainer, debugger, optimization expert, or curriculum guide — one click, no setup.

Live code sandbox

Edit HTML, CSS, and JavaScript with full syntax highlighting, watch the preview update as you type, then hit Run to execute your code.

Talk to your tutor

Prefer to talk it out? Turn on voice mode to ask questions out loud and hear the replies — typed chat is always there too.

Learn anywhere

On a phone the app becomes a chat-only tutor, so you can keep learning on the go without the desktop editor.
